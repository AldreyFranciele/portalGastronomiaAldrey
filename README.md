# 🍳 Portal Gastronomia - Receitas e Notícias



## 📖 Sobre o Projeto

O **Portal Gastronomia** é um site completo desenvolvido em PHP que permite aos usuários compartilhar e descobrir receitas e notícias sobre culinária.

- 📝 **Publicar** receitas e notícias sobre culinária
- 👤 **Gerenciar** seu perfil e publicações
- 🔍 **Explorar** receitas de outros membros
- 📧 **Enviar** mensagens de contato

 ### 👩🏻‍💻 Desenvolvedor 
 - Aldrey Franciele Da Silva Dutra

### ✨ Funcionalidades Principais

| Funcionalidade | Descrição |
|---------------|-----------|
| 🏠 **Página Inicial** | Destaque para receitas especiais e últimas publicações |
| 📰 **Receitas e Notícias** | Sistema unificado para publicar ambos os tipos de conteúdo |
| 👤 **Perfil do Usuário** | Gerenciamento de dados pessoais e senha |
| 🍳 **Dashboard** | Painel para gerenciar todas as publicações do usuário |
| 📱 **Responsivo** | Layout adaptável para mobile, tablet e desktop |
| 🔐 **Autenticação** | Login e cadastro seguros com hash de senha |
| ⚡ **Efeitos Visuais** | Animações e hover effects em toda a interface |
| 📧 **Contato** | Formulário para envio de mensagens |
